# YT Downloader App

A modern, easy-to-use desktop application to download YouTube videos in 1080p with audio, built with Python, yt-dlp, and customtkinter.

## Prerequisites
- **FFmpeg**: You must have `ffmpeg` installed to merge the 1080p video and audio streams properly. 
   - If it is not installed, open PowerShell and run: `winget install ffmpeg`

## How to use
1. Simply double-click on the **`Run.bat`** file in this folder.
2. A beautiful GUI will open up!
3. Paste your YouTube URL and click **Download Video**.

The video will be saved right in this same folder.
